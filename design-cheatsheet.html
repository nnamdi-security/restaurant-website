Do's and Don'ts
Do — Colors

Copy the exact hex from this sheet

color: #C8922A;
Don't — Colors

Never invent your own color

color: orange;  /* NO */
Do — Fonts

Use exactly the two agreed fonts

font-family: 'Playfair Display', Georgia, serif;
Don't — Fonts

Never use a third font

font-family: 'Poppins';  /* NO */
Do — Hover effects

All cards and buttons need a hover transition

transition: all 0.3s ease;
Don't — Transitions

Never use a different duration

transition: all 1s;  /* NO */
Do — Images

Fix image dimensions and crop with cover

height: 220px; object-fit: cover;
Don't — Images

Never let images stretch or auto-resize freely

width: 100%;  /* without height = NO */
Do — Section spacing

Every section must breathe equally

padding: 5rem 5%;
Don't — Section spacing

Never make sections feel cramped or uneven

padding: 10px;  /* NO */
Do — Headings

Always use Playfair Display for headings

font-family: 'Playfair Display', Georgia, serif;
Don't — Headings

Never style a heading with Lato

h2 { font-family: 'Lato'; }  /* NO */
Before you open a Pull Request — check these
Every color in my CSS is from this cheat sheet — no invented colors
I only used 'Playfair Display' for headings and 'Lato' for everything else
Every section has padding: 5rem 5% (or close to it)
All images use object-fit: cover with a fixed height
Every card and button has a transition: all 0.3s ease on hover
My page hero has a dark overlay: rgba(28,20,16,0.55)
I linked tokens.css BEFORE my own style.css in the <head>
I added class="active" to the correct nav link for my page
My page looks correct at mobile (375px), tablet (768px), desktop (1280px)
All my images have descriptive alt="..." text
